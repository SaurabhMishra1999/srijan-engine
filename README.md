# Srijan Engine

Advanced AI Text-to-Movie software that converts text scripts into 3D animated scenes.

## Tech Stack
- Python 3.14
- CustomTkinter (GUI)
- Blender (Portable, Eevee engine for rendering)

## Features
- Professional hacker-style dark theme UI
- AI-powered script analysis and scene generation
- 5-second animated clips rendered via Blender
- USB-portable setup

## Setup
1. Ensure Python 3.14 is installed.
2. Install dependencies: `pip install -r requirements.txt`
3. Place portable Blender in the `blender/` folder.
4. Run `python src/gui/main.py` to start the GUI.

## Usage
Enter a script in the text box and click "Generate Movie" to create an animated scene.
